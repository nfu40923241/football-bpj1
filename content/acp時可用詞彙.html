acp時可用詞彙

以下是一些常見用於更新網站時標註的英文詞彙：

  1. Title (標題)
  2. Heading (標題)
  3. Subheading (副標題)
  4. Label (標籤)
  5. Button (按鈕)
  6. Form (表單)
  7. Input (輸入)
  8. Select (選擇)
  9. Checkbox (勾選框)
  10. Radio button (單選按鈕)
  11. Menu (菜單)
  12. Navigation (導覽)
  13. Footer (頁尾)
  14. Link (連結)
  15. Image (圖片)
  16. Video (影片)
  17. Text (文字)
  18. Paragraph (段落)
  19. List (清單)
  20. Table (表格)
  21. Error message (錯誤訊息)
  22. Success message (成功訊息)
  23. Alert (警示)
  24. Confirmation (確認)
  25. Submit (提交)
  26. Cancel (取消)
  27. Back (返回)
  28. Next (下一步)
  29. Save (儲存)
  30. Edit (編輯)

以下是一些常見的動詞，可用於描述網站更新操作：

  1. Update (更新)
  2. Modify (修改)
  3. Add (新增)
  4. Delete (刪除)
  5. Edit (編輯)
  6. Publish (發佈)
  7. Upload (上傳)
  8. Download (下載)
  9. Move (移動)
  10. Resize (調整大小)
  11. Change (變更)
  12. Rearrange (重新排列)
  13. Customize (自訂)
  14. Refresh (重新整理)
  15. Preview (預覽)
  16. Save (儲存)
  17. Undo (復原)
  18. Redo (重做)
  19. Copy (複製)
  20. Paste (貼上)
